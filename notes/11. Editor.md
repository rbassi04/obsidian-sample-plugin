# Editor:
Exposes operations for reading and manipulating an active Markdown document in edit mode.

Can access editor in a command by using `editorCallback`[.](https://docs.obsidian.md/Plugins/Editor/Editor)

To use editor elsewhere, access it from the active view with `getActiveViewOfType` for type `MarkdownView`: 
```js
const view = this.app.workspace.getActiveViewOfType(MarkdownView);

// Make sure the user is editing a Markdown file.
if (view) {
	const cursor = view.editor.getCursor();

	// ...
}
```

## editorCallBack()
`replaceRange()` method replaces the text between two cursor positions. If you only give it one position, it inserts the new text between that position and the next.

Use `replaceSelection()` for replaced highlighted text.

```js
...
    this.addCommand({
      id: 'insert-todays-date',
      name: 'Insert today\'s date',
      editorCallback: (editor: Editor) => {
        editor.replaceRange(
          moment().format('YYYY-MM-DD'),
          editor.getCursor()
        );
      },
    });
...
```

# Markdown Read view Renderer:
The callback function of `registerMarkdownPostProcessor` is run after the HTML has been rendered. `element` is the HTML, `context` is 

```js
const ALL_EMOJIS: Record<string, string> = {
  ':+1:': '👍',
  ':sunglasses:': '😎',
  ':smile:': '😄',
};

export default class ExamplePlugin extends Plugin {
  async onload() {
		// For after the Markdown has been rendered in Read mode
    this.registerMarkdownPostProcessor((element, context) => {
			// 
      const codeblocks = element.findAll('code');

      for (let codeblock of codeblocks) {
        const text = codeblock.innerText.trim();
        if (text[0] === ':' && text[text.length - 1] === ':') {
          const emojiEl = codeblock.createSpan({
            text: ALL_EMOJIS[text] ?? text,
          });
          codeblock.replaceWith(emojiEl);
        }
      }
    });
  }
}
```
